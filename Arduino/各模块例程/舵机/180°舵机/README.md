# 180° 舵机

为了方便，以下 `180°` 舵机简称为“舵机”

## 接线方式

将 `VCC`、`GND`、`SIG` 引脚一一对应即可

## 食用方法

与 `360°` 舵机操作不同， `180°` 舵机操作需要使用相关库 `servo.h`

同时通过 `myservo.write(angle);` 设定舵机舵盘的角度，角度 `angle` 范围为 `0~180°`

## 例程食用说明

例程目录为 `./Servo180`，直接打开 [`./Servo180/Servo180.ino`](./Servo180/Servo180.ino) 并烧入

舵机会在 `0~180°` 间摆动
